# Changelog

All notable changes to RepLog are documented here. The format is based on Keep a Changelog, and this project adheres to Semantic Versioning.

## [1.1.0] - 2026-06-23

### Added
- Live workout logger: log sets in real time with a rest timer, plate calculator, RPE, warm-up sets, and previous-set placeholders that auto-fill on confirm.
- Routines and templates for repeatable sessions.
- Scored strength-imbalance coach: a 0-100 balance score with ring gauges and prioritised recommendation cards.
- Progress graphs and per-exercise history.
- Settings page: units (kg/lb), default rest, plate inventory, and a Stable/Beta release-channel toggle.
- Custom exercises that persist and rejoin the searchable catalog.
- History view of past workouts.

## [1.0.0] - 2026-06-22

### Added
- Import workout notes via screenshot OCR, pasted text, or screen-recording.
- On-device parser with an exercise catalog; auto-categorisation by split and date.
- All-time bests and per-exercise progress pages.
- Installable, offline-capable PWA. 100% on-device - your data never leaves the browser.
